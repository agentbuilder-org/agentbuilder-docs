---
title: "Create Thread"
api: "POST /api/threads"
---

Creates a new conversation thread with the specified agent.

<ParamField body="agentName" type="string" required>
  The name of the agent to use for this thread. Must match an agent defined in `agents/agents/`.
</ParamField>

<ParamField body="userId" type="string">
  Optional user identifier for tracking and authorization.
</ParamField>

<ResponseField name="id" type="string" required>
  Unique thread identifier (UUID)
</ResponseField>

<ResponseField name="agentName" type="string" required>
  The agent name associated with this thread
</ResponseField>

<ResponseField name="created_at" type="number" required>
  Unix timestamp in milliseconds
</ResponseField>

<ResponseField name="metadata" type="object">
  Thread metadata object
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://your-worker.workers.dev/api/threads \
  -H "Content-Type: application/json" \
  -d '{
    "agentName": "support_agent",
    "userId": "user_abc123",
    "metadata": {
      "source": "web"
    }
  }'
```


```typescript JavaScript
const response = await fetch('https://your-worker.workers.dev/api/threads', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    agentName: 'support_agent',
    userId: 'user_abc123',
    metadata: { source: 'web' }
  })
});
const thread = await response.json();
```

</RequestExample>
